Alignment & Synteny Parameters

These parameters apply to the alignment, anchor filtering/loading, and synteny computation for a pair of projects.

If you want to re-use parameters from a previous pair, first close the Parameters window, then select the project pair in the "Available Syntenies" table, and then re-open the Parameters window and use the "Load Previous" button.

Parameter Description Stage Default Value
Top N Retain the top N hits for every sequence region (as well as all hits with score at least 80% of the Nth). Anchor Load 2
Min Dots Minimum number of anchors required to define a synteny block. Synteny 7
Merge Blocks Merge overlapping (or nearby) synteny blocks into larger blocks. Synteny Off
No Overlapping Blocks If two synteny blocks overlap on a chromosome, keep only the larger one. Synteny Off
NUCmer Only User NUCmer for all alignments.* Align Off
PROmer Only User PROmer for all alignments.* Align Off
BLAT Args Arguments for BLAT (FPC alignments only) Align -minScore=30 -minIdentity=70 -tileSize=10 -qMask=lower -maxIntron=10000

* Note that, by default, PROmer is used for alignments between different projects, while NUCmer is used for self alignments.
