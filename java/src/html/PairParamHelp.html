Alignment & Synteny Parameters

These parameters apply to the alignment, anchor processing, and synteny computation (AAS) for a pair of projects. Typically, the defaults will work fine.

The arguments for MUMmer are NOT checked for correctness.

Parameter Description Stage Default
Top N Retain the top N hits for every sequence region (as well as all hits with score at least 80% of the Nth). Anchor Load 2
Min Dots Minimum number of anchors required to define a synteny block. Synteny 7
Merge Blocks Merge overlapping (or nearby) synteny blocks into larger blocks. Synteny Off
No Overlapping Blocks If two synteny blocks overlap on a chromosome, keep only the larger one. Synteny Off
PROmer Only Use PROmer for all alignments.1 Align Off
NUCmer Only Use NUCmer for all alignments.1 Align Off
PROmer Args Arguments for PROmer Align  
NUCmer Args Arguments for NUCmer Align  
Self Args When aligning a chromosome to itself.2 Align  
FPC BLAT Args Arguments for BLAT (FPC alignments only) Align See 3

1 By default, PROmer is used for alignments between different projects, while NUCmer is used for self alignments.

2 When self-alignment is performed, standard arguments are used when comparing different chromosomes. However, additional arguments may be desired when a chromosome sequence is run against itself, e.g. --nosimplify or --maxmatch, though the latter can take a very long time.

3 FPC BLAT Args will only be shown if there is a data/fpc directory. The defaults are -minScore=30 -minIdentity=70 -tileSize=10 -qMask=lower -maxIntron=10000. The arguments are NOT checked for correctness.

Rules for saving parameters

If parameters were set, the parameters will be shown on the summary page.

MUMmer parameters

To see the parameters for the default MUMmer V3 on MacOS: 
./ext/mummer/mac/mummer -h
./ext/mummer/mac/promer -h
./ext/mummer/mac/nucmer -h
To see the parameters for the default MUMmer V3 on Linux: 
./ext/mummer/linux/mummer -h
./ext/mummer/linux/promer -h
./ext/mummer/linux/nucmer -h
If you compiled V4 in the /ext directory: 
./ext/mummer4/m4/bin/mummer -h
./ext/mummer4/m4/bin/promer -h
./ext/mummer4/m4/bin/nucmer -h
