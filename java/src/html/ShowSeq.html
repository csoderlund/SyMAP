Show Seq Options

HitThe clustered hits will be listed in a FASTA type output. If the hit is on the negative strand, it is reverse complemented.
Reverse HitThe hit will be shown opposite of its strand, i.e. if the hit is on the positive strand, it will be reversed complemented; otherwise, it will not be.
Full HitThe full sequence from start to end will be displayed; this is relevant for hits with clustered hits.
RegionThe exact region selected.
