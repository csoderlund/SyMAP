FPC Project Parameters

This page provides all the setup parameters for an FPC project, as follows:.

fpc_file

Browse to the FPC file you wish to load.

bes_files

Select one or more fasta files containing BAC-end sequences for the FPC clones. Note that the BES must be named exactly like the FPC clones, plus suffixes ".r" and ".f" showing the strand. BES which can't be matched to a clone will be ignored.

marker_files

Select one or more fasta files containing marker sequences. The marker names must match exactly with the names of markers in the FPC project.

grp_prefix

If provided, this allows SyMAP to remove the prefix from your group/chromosome names, and use the remaining part as a shorter display name (e.g., "5" instead of "chr5").

grp_order

If you have a mixture of numeric and alphabetic anchoring group names, you can simply list them here in the desired order. This overrides grp_sort.

cbsize

The approximate size of a CB (consensus band) unit, in basepairs. This is used to draw the FPC contigs to scale against other sequences.

description

Description of the project. Currently, this is only shown in the Project Manager Summary section.

category

Category label for the project, currently only used in the Summary list of the Project Manager window.
