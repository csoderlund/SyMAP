Explain Summary

Clustered hits have n≥1 MUMmer hits that may have gaps between them or overlap.
Subhit is a MUMmer hit that is part of a clustered hit.
Synteny Block is composed of clustered hits.

Abbrev: Gene (GN), Clustered hit (CH), SubHit (SH), Synteny Block (SB), Cluster hit in block (CHB)
If the second project does not have a value, then it is the same for the two projects.
